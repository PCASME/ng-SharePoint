SharePoint
service in module ngSharePoint

Description

Provides top level access to SharePoint web sites api. Through this provider it is possible to access to any SharePoint web.

Dependencies

  • SPUtils
  • SPWeb

Methods

  • get(webServerRelativeUrl, queryUrl)

    Returns the results of the query to the SharePoint API REST specified by the required url.

    Parameters
    ParamTypeDetails
    webServerRelativeUrlstring

    The web server relative url of the web where to perform the query.

    queryUrlstring

    The url of the complete query to perform. It refers to the query part after //_api/web/...

    Returns
    promise

    Promise with results of the query.

    Example

    SharePoint.get("/sites/rrhh/_api/web/Lists/GetByTitle('MyList')/RoleAssignments/GetByPrincipalId(35)").then(function(results) {
  // ... do something with the 'rrhh' web object
});

  • getCurrentWeb()

    Returns an SPWeb object initialized with the current SharePoint web. That means, the web context where this sentence is executed

    Returns
    promise

    Promise with a new SPWeb object that allows access to web methods and properties

    Example

    SharePoint.getCurrentWeb().then(function(web) {
  // ... do something with the web object
});

  • getWeb(url)

    Returns the SPWeb specified by the required url

    Parameters
    ParamTypeDetails
    urlstring

    The url of the web that you want to retrieve

    Returns
    promise

    Promise with a new SPWeb object that allows access to web methods and properties

    Example

    SharePoint.getWeb('/sites/rrhh').then(function(web) {
  // ... do something with the 'rrhh' web object
});
